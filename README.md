## Installation

Clone the repository:

~~~bash
git clone https://github.com/AdrianRo147/dsa-library
~~~

## Usage

To build using CMake you can do so using following commands

```
cmake -B build
cmake --build build
```

To run tests you can start output binary

```
./build/out
```
